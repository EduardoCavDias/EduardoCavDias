## 🚀 Projeto em Destaque

<div align="center">

### 🎯 PONTUX — Sistema de Registro de Presença Digital

</div>

> **PWA completo em produção**, desenvolvido do zero para a gestão de presença de jovens aprendizes.

```
🌐 Live:  https://sistema-presenca-curso-abcvida.web.app
🔧 Stack: HTML5 · CSS3 · JavaScript ES2022 · Firebase Auth + Firestore + Hosting
📱 PWA:   Funciona offline, instalável no celular, responsivo
🔒 Auth:  Login com bloqueio por dispositivo (deviceID + fingerprint)
```

**O que o Pontux faz:**
- 📍 Registro de ponto com **verificação de localização GPS** (raio configurável)
- 👤 Controle de presença individual por aprendiz e turma
- 📊 **Dashboard administrativo** com gráficos de frequência em tempo real
- 🔄 Sistema de **troca de dias** entre aprendizes com aprovação do admin
- 📅 Gerenciamento de feriados, turmas e configurações do sistema
- 🛡️ Regras de segurança Firestore contra fraudes
- 📤 **Exportação de relatórios** em CSV
